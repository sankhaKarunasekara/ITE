Improve Travel Experience through OLBS in Horton Plains

ITE is an offline system to improve traveler experience in offline environment. Study area of this application is around Horton Plains Sri Lanka. Application mainly used location matching algorithm. Application contains special location information, trails, roads and cliff data. We provide voice information and display messages. Voice messages is the very interesting part of this application hence it is become a virtual guide. When someone with that application reaches a particular location, application will identify the location and automatically play a voice message. It explains information about the location reached. If the traveler needs more information, the display contains more information about the location. When using trails, road and cliff information together with GPS sensor data, it can identify safe path. And when using that information, it can navigate safely in all-weather conditions.
